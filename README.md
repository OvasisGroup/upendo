## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.

## PWA Features

This application is configured as a Progressive Web App (PWA), which means:

- **Installable**: Users can install the app on their devices
- **Offline Support**: The app works offline using service workers
- **App-like Experience**: Runs in standalone mode on mobile devices

### PWA Configuration

- Manifest file: `public/manifest.json`
- Icons: Generated from `public/icon.svg`
- Service Worker: Auto-generated by `next-pwa`

### Regenerating Icons

If you need to update the app icon:

1. Edit `public/icon.svg` with your new design
2. Run: `node scripts/generate-icon-placeholders.cjs`

This will regenerate all required icon sizes (192x192, 384x384, 512x512) and the favicon.
